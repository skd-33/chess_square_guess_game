# Chess Square Guessing Game
 **Play here:**  
 https://skd-33.github.io/chess_game/
 
## How to Play

1. Choose a **time control** (3s / 5s / 10s)
2. Click **Start Game**
3. A random square (e.g. `E4`) is shown
4. Click the **correct square on the board**
5. You have **3 lives**
6. Wrong click or timeout = lose 1 life
7. Game ends when lives reach 0
